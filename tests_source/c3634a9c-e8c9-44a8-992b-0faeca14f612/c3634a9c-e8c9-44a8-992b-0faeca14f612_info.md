## Overview
This test evaluates endpoint protection against the Bring Your Own Vulnerable Driver (BYOVD) attack chain employed by Akira ransomware. The test simulates the sophisticated multi-stage attack that combines legitimate vulnerable drivers with malicious drivers to achieve privilege escalation and disable Windows Defender security controls.

## Technique Details
- **MITRE ATT&CK**: T1068 (Exploitation for Privilege Escalation), T1562.001 (Impair Defenses: Disable or Modify Tools)
- **Tactic**: Privilege Escalation, Defense Evasion
- **Platform**: Windows
- **Permissions Required**: Administrator
- **Category**: Ransomware Simulation, BYOVD Attack, Defense Evasion

## Attack Chain Description

### Stage 1: Driver Deployment (BYOVD Technique)
The test simulates dropping two drivers:
- **rwdrv.sys** (15,360 bytes): Legitimate ThrottleStop driver with known vulnerabilities
- **hlpdrv.sys** (24,576 bytes): Malicious helper driver used for defense evasion

Both drivers are dropped to the `C:\F0` directory, following F0RT1KA testing standards.

### Stage 2: Service Registration
Creates Windows kernel services to load the drivers:
- **mgdsrv**: "Malicious Guard Driver Service" pointing to rwdrv.sys
- **KMHLPSVC**: "Kernel Mode Helper Service" pointing to hlpdrv.sys

### Stage 3: Defense Evasion Execution
Executes a PowerShell script that:
- Bypasses execution policy restrictions
- Attempts to modify Windows Defender registry policies
- Targets `HKLM\SOFTWARE\Policies\Microsoft\Windows Defender\DisableAntiSpyware`
- Implements tamper protection bypass techniques

## Purpose

This test validates that a Windows endpoint meets the CIS Benchmark Level 1 Profile security hardening requirements. Level 1 represents the minimum security configuration recommended for any enterprise system. The 52 controls cover:

1. **Password and credential policy** enforcement
2. **Account management** best practices
3. **Network authentication** protocol hardening
4. **Credential protection** mechanisms (LSASS, WDigest, Credential Guard)
5. **Windows Firewall** configuration
6. **Audit and logging** policy completeness
7. **Endpoint protection** features (ASR rules, RDP, BitLocker)

## Bundled Validators

### 1. Credential & Password Policy (10 controls)
Validates password and lockout policy using secedit security policy export:
- CH-CW1-001: Password History >= 24 passwords
- CH-CW1-002: Maximum Password Age <= 365 days
- CH-CW1-003: Minimum Password Age >= 1 day
- CH-CW1-004: Minimum Password Length >= 14 characters
- CH-CW1-005: Password Complexity enabled
- CH-CW1-006: Relax Minimum Password Length Limits
- CH-CW1-007: Account Lockout Duration >= 15 minutes
- CH-CW1-008: Account Lockout Threshold <= 5 attempts
- CH-CW1-009: Reset Lockout Counter >= 15 minutes
- CH-CW1-010: Administrator Account Lockout enabled

### 2. Account Management (8 controls)
Validates account configuration and LAPS deployment:
- CH-CW1-011: Block Microsoft Accounts
- CH-CW1-012: Guest Account Disabled
- CH-CW1-013: Administrator Account Renamed
- CH-CW1-014: Guest Account Renamed
- CH-CW1-015: LAPS Enabled (Windows LAPS or Legacy)
- CH-CW1-016: LAPS Password Complexity >= 4
- CH-CW1-017: LAPS Password Length >= 15
- CH-CW1-018: LAPS Password Age <= 30 days

### 3. Network Authentication Hardening (8 controls)
Validates authentication protocol security:
- CH-CW1-019: NULL Session Fallback Disabled
- CH-CW1-020: PKU2U Authentication Disabled
- CH-CW1-021: Kerberos AES-Only Encryption

- CH-CW1-023: NTLM Minimum Session Security (NTLMv2 + 128-bit)
- CH-CW1-024: SMBv1 Disabled
- CH-CW1-025: SMB Signing Required (Server)
- CH-CW1-026: SMB Signing Required (Client)

### 4. Credential Protection (3 controls)

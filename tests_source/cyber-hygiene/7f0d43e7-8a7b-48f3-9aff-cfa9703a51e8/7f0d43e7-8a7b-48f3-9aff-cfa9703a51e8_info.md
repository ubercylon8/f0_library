## Description

This bundle validates 35 CIS Linux Benchmark Level 1 controls organized across 7 security categories. It uses the multi-binary architecture where each category compiles as a separate embedded binary, providing quarantine resilience -- if an EDR agent quarantines one validator, the remaining validators continue executing and reporting.

The controls cover the essential Level 1 hardening baseline that CIS recommends for all Linux systems, including filesystem hardening, service minimization, network security, audit logging, SSH configuration, access control, and system maintenance.

## Detection Opportunities

1. **Filesystem Hardening** - cramfs/USB module status, mount options
2. **Service Minimization** - Unnecessary service detection
3. **Network Security** - Sysctl values, firewall policy enforcement
4. **Audit Logging** - auditd coverage of identity files, permissions, privileged commands
5. **SSH Hardening** - Root login, authentication method, access restrictions
6. **Access Control** - sudo policy, PAM configuration, password policy
7. **System Integrity** - File permissions, ownership, SUID/SGID audit

## Expected Results

### Compliant System (Code 126)
- All 35 controls pass

## Purpose

This bundled test validates the security posture of Windows endpoints by checking 10 critical security configurations in a single execution. It consolidates individual cyber-hygiene validators into one binary to:

1. Reduce deployment complexity (single binary instead of 10)
2. Provide comprehensive security posture assessment
3. Enable baseline compliance verification
4. Support CIS Benchmark validation

## Bundled Validators

### 1. Microsoft Defender Configuration
Validates core Defender features:
- Real-time Protection
- Behavior Monitoring
- Tamper Protection
- Cloud-delivered Protection (MAPS)
- Automatic Sample Submission
- Potentially Unwanted Application (PUA) Protection

### 2. LSASS Protection
Validates credential protection mechanisms:
- RunAsPPL (Protected Process Light)
- Credential Guard
- Virtualization-Based Security (VBS)

### 3. Attack Surface Reduction Rules
Validates 8 critical ASR rules:
- Block executable content from email/webmail
- Block Office apps from creating child processes
- Block Office apps from creating executable content
- Block Office apps from code injection
- Block JavaScript/VBScript launching executables
- Block obfuscated scripts
- Block Win32 API calls from Office macros
- Block Office communication apps from creating child processes

### 4. SMB Hardening
Validates SMB security configuration:
- SMBv1 disabled
- SMB Server signing required
- SMB Client signing required
- SMB Encryption enabled
- Null session restrictions

### 5. PowerShell Security
Validates PowerShell logging and restrictions:
- Script Block Logging
- Module Logging
- Transcription
- Constrained Language Mode policy

### 6. Network Protocol Hardening
Validates network protocol security:
- LLMNR disabled
- NetBIOS over TCP/IP disabled
- WPAD disabled
- IPv6 tunneling technologies disabled

### 7. Windows Audit Logging
Validates 9 critical audit categories:
- Logon events
- Logoff events
- Account lockout
- Special logon
- Process creation
- Audit policy changes
- User account management
- Computer account management
- Security group management

### 8. Account Lockout Policy
Validates CIS-compliant password policies:
- Lockout threshold (≤5 attempts)
- Lockout duration (≥15 minutes)
- Reset counter (≥15 minutes)
- Minimum password length (≥14 characters)
- Password complexity enabled

### 9. LAPS (Local Administrator Password Solution)
Validates LAPS configuration:
- Windows LAPS (built-in, Server 2019+)
- Legacy LAPS (Microsoft LAPS)

### 10. Print Spooler Hardening
Validates PrintNightmare (CVE-2021-34527) mitigations:
- Print Spooler service status
- Point and Print restrictions

## Exit Code Logic

| Code | Condition | Meaning |
|------|-----------|---------|
| **126** | All 10 validators pass | Endpoint is properly hardened |
| **101** | Any validator fails | Security gaps detected |
| **999** | Admin check fails | Test prerequisites not met |

## Overview
This test simulates AMSI (Anti-Malware Scan Interface) bypass techniques using TrollDisappearKey, a tool that manipulates Windows registry queries to prevent AMSI provider DLLs from loading properly. This enables the execution of malicious .NET assemblies without triggering AMSI scanning mechanisms. The test uses Seatbelt as a test payload to demonstrate successful AMSI bypass.

## Technical Implementation
TrollDisappearKey works by:
- Hooking the `RegOpenKeyExW` Windows API function
- Intercepting registry queries for "Software\\Microsoft\\AMSI\\Providers"
- Redirecting queries to "Software\\Microsoft\\AMSI\\Providers " (with trailing space)
- Breaking the AMSI provider DLL loading mechanism
- Downloading and executing Seatbelt.exe to demonstrate successful AMSI bypass
- Modified to execute once and exit cleanly (no interactive loop)

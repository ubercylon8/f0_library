## Description

This test simulates the sophisticated UAC bypass and defense evasion techniques used by SafePay malware. The test leverages the CMSTPLUA COM interface, a known UAC bypass method that abuses auto-elevated COM objects to execute code with elevated privileges without triggering UAC prompts.

## Technical Details

### UAC Bypass Mechanism
The test exploits the CMSTPLUA COM interface (GUID: {3E5FC7F9-9A51-4367-9063-A120244FBEC7}) which is auto-elevated in Windows. This interface can be abused to execute arbitrary code with elevated privileges without user interaction.

### Persistence Mechanism
Creates a registry autorun entry at:
- **Path**: `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`
- **Name**: `SafePayService`
- **Value**: `6F22-C16F-0C71-688A`

### Defense Evasion
Attempts to disable Windows Defender through GUI automation, simulating the behavior of SafePay malware that tries to impair security tools.

## Test Execution Flow

1. **Initialization**: Drops PowerShell script to `c:\F0\safepay_uac_bypass.ps1`
2. **Execution Policy Bypass**: Runs script with `-ExecutionPolicy Bypass`
3. **Privilege Check**: Verifies current privilege level
4. **UAC Bypass**: Attempts to create and use CMSTPLUA COM object
5. **Persistence**: Creates registry autorun entry
6. **Defense Evasion**: Simulates Windows Defender disablement
7. **Verification**: Checks if registry persistence was successful

## Detection Opportunities

Security solutions should detect:
- Creation of CMSTPLUA COM object ({3E5FC7F9-9A51-4367-9063-A120244FBEC7})
- Suspicious PowerShell execution with bypass parameters
- Registry modification at CurrentVersion\Run
- Attempts to interact with Windows Security GUI
- Process creation with elevated privileges without UAC prompt

## Expected Results

### Protected System (EDR/AV Active)
- **Exit Code**: 126 (Endpoint.ExecutionPrevented)
- UAC bypass attempt blocked
- Registry persistence prevented
- PowerShell script quarantined or execution blocked

### Unprotected System
- **Exit Code**: 101 (Endpoint.Unprotected)
- Registry persistence successfully created
- UAC bypass potentially successful
- No alerts or blocks from security software

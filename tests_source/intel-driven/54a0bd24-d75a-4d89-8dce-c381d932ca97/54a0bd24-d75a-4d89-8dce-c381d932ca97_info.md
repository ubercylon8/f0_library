## Description

This test simulates the dominant persistence and evasion technique used by sophisticated Linux malware: LD_PRELOAD shared library hijacking combined with PAM credential hooking. It models real-world attack chains from four distinct threat actors/malware families:

- **Perfctl** (eCrime, cryptomining): Deploys libgcwrap.so that hooks pam_authenticate for credential capture and pcap_loop for traffic hiding. Uses XOR encryption with key 0xAC. Targeted millions of Linux servers for 3+ years with trojanized crontab binaries, systemd services, and ~/.profile modifications.
- **Symbiote** (eCrime, financial sector, Latin America): Hooks libc + libpcap to hide files, processes, and network artifacts. 2025 variants added IPv6 and 8-port support across TCP/UDP/SCTP.
- **Auto-Color** (universities, government): Writes malicious library to /etc/ld.preload and hooks open() to scrub /proc/net/tcp entries, preventing detection of C2 connections.
- **WolfsBane** (Gelsemium APT, state-sponsored): Modified BEURK userland rootkit that hooks open, stat, and readdir via /etc/ld.so.preload.

The test creates simulated artifacts across 5 phases covering the complete attack lifecycle from initial persistence through credential harvesting and network concealment.

## Technical Details

### Attack Flow

1. **Phase 1: LD_PRELOAD Injection (T1574.006)**
   - Creates simulated malicious .so file (libgcwrap.so) with ELF header and hook stubs
   - Generates /etc/ld.preload content (Auto-Color pattern)
   - Generates /etc/ld.so.preload content (WolfsBane pattern)
   - Creates systemd service with LD_PRELOAD environment variable
   - Simulates ~/.profile injection for login persistence

2. **Phase 2: PAM Credential Hooking (T1003.008)**
   - Simulates pam_authenticate hook capturing credentials from sshd, sudo, and login
   - Creates /etc/shadow dump with SHA-512 and yescrypt hashes
   - Enumerates /etc/passwd including Perfctl's trailing-space nologin bypass
   - Generates credential capture log mimicking Perfctl's exfiltration staging

3. **Phase 3: Network Artifact Hiding (T1014)**
   - Creates original /proc/net/tcp with all network connections
   - Creates rootkit-scrubbed version hiding 3 C2 connections
   - Documents Symbiote's libpcap pcap_loop filtering rules
   - Reports hidden connections with attribution to specific rootkit hooks

4. **Phase 4: XOR-Encrypted C2 Config (T1059.004)**
   - Generates plaintext C2 configuration with pool/wallet/beacon settings
   - XOR-encrypts config with Perfctl's key 0xAC
   - Verifies encryption round-trip
   - Creates trojanized crontab showing hidden vs. visible entries

5. **Phase 5: SUID Abuse (T1548.001)**
   - Simulates `find / -type f -perm -u=s` enumeration
   - Identifies 6 exploitable SUID binaries from GTFOBins
   - Documents exploitation vectors (find -exec, vim -c, bash -p)
   - Simulates SUID-based privilege escalation to root

### Key Indicators

- ELF-format .so files with hook function stubs (pam_authenticate, pcap_loop, open, stat, readdir)
- Files named libgcwrap.so, etc_ld.preload, etc_ld.so.preload
- XOR-encrypted binary blobs (high entropy, key 0xAC)
- Credential logs with PAM service names (sshd, sudo, login)
- Shadow dump files with $6$ and $y$ hash prefixes
- SUID enumeration results with GTFOBins annotations
- Differential /proc/net/tcp views showing hidden C2 connections

## Detection Opportunities

1. **File System Activity**
   - Write operations to /etc/ld.preload or /etc/ld.so.preload
   - New .so files appearing in /lib/ or /usr/lib/ directories
   - systemd service files with LD_PRELOAD environment variables
   - ~/.profile modifications with export LD_PRELOAD

2. **Credential Access Indicators**
   - Abnormal access to /etc/shadow by non-root processes
   - PAM module modifications or new PAM configuration files
   - Credential log files in unexpected locations
   - Bulk authentication event correlation

3. **Network Concealment**
   - Discrepancy between /proc/net/tcp and kernel-level connection data
   - eBPF-based network monitoring bypassing userland hooks
   - Comparing netstat output with iptables conntrack table
   - Unexpected high-entropy data in .so file sections

4. **Privilege Escalation**
   - find commands with -perm -u=s arguments
   - chmod u+s on system binaries (bash, sh)
   - Processes running with mismatched real/effective UIDs
   - GTFOBins command patterns (find -exec, vim -c, nmap --interactive)

5. **Persistence Mechanisms**
   - New systemd services targeting multi-user.target
   - Crontab binary integrity validation
   - ~/.profile/~/.bashrc modifications
   - Watchdog processes monitoring persistence removal

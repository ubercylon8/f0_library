### Key Disclosures Modeled

From the GTIG report, HONESTCUE is described as:

> "In September 2025, GTIG observed malware samples tracked as HONESTCUE, leveraging Gemini's API to outsource functionality generation. HONESTCUE is a downloader and launcher framework that sends a prompt via Google Gemini's API and receives C# source code as the response. HONESTCUE shares capabilities similar to PROMPTFLUX's 'just-in-time' technique; however, rather than leveraging an LLM to update itself, HONESTCUE calls the Gemini API to generate code that operates the 'stage two' functionality."

The test models the three behaviors GTIG identifies as core to HONESTCUE's operation:

1. **Gemini API prompt/response at runtime** — modeled by stage 1 via loopback mock Gemini server with Gemini-schema request and response JSON.
2. **C# source code as LLM response** — modeled via synthetic C# payload returned by the mock server, handed off to stage 2.
3. **Stage-two execution via compiled-and-loaded C# assembly** — modeled by stage 2 via `powershell.exe` + `System.CodeDom.CSharpCodeProvider` (`GenerateInMemory=true`) + `Assembly.Load(byte[])`.

The Discord CDN abuse in stage 3 extends the modeled chain to reflect HONESTCUE's reported (and broader industry-reported) use of trusted cloud infrastructure for follow-on payload staging.
